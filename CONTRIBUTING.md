# Contributing to Multimodal RAG

Thank you for your interest in contributing! Since this is an academic final project, direct contributions are limited, but suggestions are welcome.

## How to Contribute

1. **Fork the repository**
2. **Clone your fork**:
    ```bash
    git clone https://github.com/your-username/repo-name.git
    ```
3. **Create a feature branch**:
    ```bash
    git checkout -b feature/AmazingFeature
    ```
4. **Commit your changes**:
    ```bash
    git commit -m 'feat: Add some AmazingFeature'
    ```
5. **Push to the branch**:
    ```bash
    git push origin feature/AmazingFeature
    ```
6. **Open a Pull Request**

## Reporting Bugs
Please use the GitHub Issues tab to report bugs. Include your GPU specs and VRAM usage in the report.
